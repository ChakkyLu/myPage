HTML5 UP is a side project of AJ’s (ie. me). I started it as a way to both test my skel.js framework (formerly 5grid) and sharpen up my design skills a bit.

skel.js? WTF is that?

skel.js is a lightweight framework for building responsive sites and apps. All of my designs at HTML5 UP (including this one) are built on this framework.
